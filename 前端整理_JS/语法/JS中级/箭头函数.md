### 箭头函数格式

~~~
        let fn = (a, b) => {
            return a + b;
        }
        // 注意：如果函数体只有一行代码，那么可以省略大括号，
        // 如果把打括号省略的话，会自动的返回结果，不需要写return
        let fn2 = a => a * a;
        console.log(fn2(6));
~~~

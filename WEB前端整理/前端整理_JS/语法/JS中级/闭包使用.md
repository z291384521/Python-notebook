### 闭包解释

闭包：一个作用域有权访问另外一个作用域的局部变量
作用：延申变量的使用范围

~~~
<script>
  function foo() {
    let i = 0;

    // 函数内部分函数
    function bar() {
			console.log(++i);
    }

    // 将函数做为返回值
    return bar;
  }
  
  // fn 即为闭包函数
  let fn = foo();
  
  fn(); // 1
</script>
~~~

总结：

闭包：一个作用域有权访问另外一个作用域的局部变量，

好处：可以把一个变量使用范围延伸

1. 闭包本质仍是函数，只不是从函数内部返回的
2. 闭包能够创建外部可访问的隔离作用域，避免全局变量污染
3. **过度使用闭包可能造成内存泄漏**

注：回调函数也能访问函数内部的局部变量。
